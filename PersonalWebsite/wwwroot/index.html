Welcome to my website

This is a personal site for projects. A space for me to learn and develop skills, as well as display my credentials. Over time I will populate it with different projects i'm working on. The site uses Microsoft Authentication, there is currently no guest tier so you will need to ask me to add you to my tenant. The site is structured as a Wiki, for no other reason than I wanted an excuse to improve my API skills, so most of the content on this site is called from a GithubRepo! If you login, most of the content is also editable. It's also a Progressive Web App(PWA), so feel free to instal locally! I have been working on a ChatBot which can take the content from this site as context and answer questions about it, however this requires a login as it costs me money!
